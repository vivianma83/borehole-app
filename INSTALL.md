# 钻孔柱状图 · 安装指南

> 当前为**未签名测试版**，首次安装需绕过系统警告。正式版上线时会移除这些步骤。

## macOS 安装

### 1. 下载安装包

拿到 `钻孔柱状图_x.y.z_aarch64.dmg`（或 `_x64.dmg`，Intel Mac）。

### 2. 挂载并复制到应用程序

- 双击 `.dmg` 打开
- 把里面的「钻孔柱状图」app 图标拖到「Applications」文件夹

### 3. 首次打开（关键步骤）

**直接双击会被系统拦住**，显示「无法打开，因为它来自未知开发者」。正确做法：

1. 打开「应用程序」(Applications) 文件夹
2. **右键点击**「钻孔柱状图」app（或按住 Control 点击）
3. 选择「打开」
4. 弹窗显示「macOS 无法验证此 App 不含恶意软件」→ 点 **"打开"**（不是默认的"完成"）
5. App 启动，从此双击就能正常打开

> 如果右键菜单里没有「打开」选项，先到「系统设置 → 隐私与安全性」滚到底部，应该能看到一个"仍要打开"按钮。

### 4. 卸载

把 `/Applications/钻孔柱状图.app` 拖到废纸篓即可。

## Windows 安装

### 1. 下载安装包

拿到 `钻孔柱状图_x.y.z_x64-setup.exe`（NSIS 安装器）或 `_x64_en-US.msi`（MSI 包）。

### 2. 首次安装（绕过 SmartScreen）

双击 `.exe` 时 Windows Defender 会弹「Windows 已保护你的电脑」蓝色窗口：

1. 点窗口中间的 **「更多信息」**
2. 出现 **「仍要运行」** 按钮，点它
3. 走过普通的安装向导（同意 → 选目录 → 安装）
4. 桌面和开始菜单出现"钻孔柱状图"快捷方式

### 3. 卸载

控制面板 → 程序和功能 → 找到"钻孔柱状图" → 卸载

## 第一次启动

App 打开后看到的应该是：
- 1400 × 900 的窗口，标题"钻孔柱状图"
- 顶栏 7 个按钮：加载示例 / 从空白开始 / 打开 .json / 保存 .json / ✏ 编辑数据 / 导出 SVG / 打印 / PDF
- 主区：空白 24 列骨架

**快速试用**：点 **加载示例** → 看到完整的 ZK1042 钻孔图 → 点 **✏ 编辑数据** → 5 个 Tab 都能改。

## 系统要求

| 平台 | 最低版本 |
|---|---|
| macOS | 10.13 (High Sierra) |
| Windows | Windows 10 1809+（需要 WebView2 Runtime） |

**Windows 用户特别提醒**：Win10/11 一般预装了 WebView2 Runtime。如果安装时提示缺失，installer 会自动联网下载。**断网环境**请用离线版（如果分发了 `*-setup-offline.exe`）。

- 当前版本**不自动保存**——你需要主动点"保存 .json"
- 数据保存为单个 `.json` 文件，可发邮件、放 U 盘、提交 Git
- 双击 `.json` 不会用 App 打开（未来版本可能注册文件关联）

## 常见问题

### 打不开 / 闪退

1. 看任务管理器/活动监视器有没有"钻孔柱状图"进程
2. 删除 `~/Library/Logs/com.synxdata.borehole/`（macOS）或 `%LOCALAPPDATA%\com.synxdata.borehole\`（Win）
3. 重新打开

### 性能慢 / 卡顿

- 100+ 行回次记录滚动时如果掉帧，是已知问题，等后续版本加虚拟列表
- macOS Intel + 老机器可能慢于 Apple Silicon

### 字体显示乱码 / □□□

System 缺中文字体（极少见）。装个思源黑体或苹方就解决。
